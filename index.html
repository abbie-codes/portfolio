Projects

Custom Payment Card Website

 Created for Code First Girls: Introduction to Web Development Course.

For this project I created responsive landing page and product build page for customisable payment cards using Javascript, HTML and CSS . This website is two pages, but the second page has multiple sections that display, or hide when you navigate using the navigation buttons. On the front page I have built a slideshow of cards (that I created using Photoshop). On the build page I have created a preview feature, when you click the option you would like to display on the card the preview section updates. I have included a form and once the form is submitted another section pops up to confirm the order. The order number rises with each order.
To view the code please visit the Github repository.

Landing Page

 Created for Udacity as part of a Front End Development Nanodegree.

For this project I created responsive landing page for a study app using Javascript, HTML and CSS . This website is one page. The navigation bar highlights the section displayed in the viewport and hides when scrolling. The section that is being viewed becomes active, highlights and an animation is added. When a user scrolls down the page a button appears to allow the user to scroll to top. All sections are collapsible.
To view the code please visit the Github repository.
