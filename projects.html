Next Adventure App

Created for Udacity as part of a Front End Development Nanodegree.

For this project I built a travel planner application. You can enter the dates and location of your trip and using Geonames API and Weatherbit API, the app with either tell you the weather forecast for your trip (if the trip is within 16 days) or the climate normals for your trip. I have also included a tickable packing list feature, a countdown to the trip and a photo of the location generated from Pixabay API. This project uses a Node.js server to store project data, Express for routing, Webpack to build the project, SCSS for styling and Javascript for functionality. The HTML and CSS is my own.


To view the code please visit the Github repository.

Opinion Miner

Weather Journal App

Created for Udacity as part of a Front End Development Nanodegree.

For this project I created a node.js server to store data from a combination of user input on a web page, the date generated in the app.js file and the current temperature (in Farenheit) which was generated using an API from OpenWeatherMap, the API requires a post code which is entered on the webpage. I used Express, cors and body-parser node.js modules in this project. This web project showcases asyncronous fetch, get and post javascript functions, and the use of an external API; a node.js server, HTML & CSS styling. I used Digital Ocean to deploy this website as they host backend node.js servers.
To view the code please visit the Github repository.

Custom Payment Card Website

 Final Project for Code First Girls: Introduction to Web Development Course.
This project won the contest for best project in the course!

For this project I created responsive landing page and product build page for customisable payment cards using Javascript, HTML and CSS . This website is two pages, but the second page has multiple sections that display, or hide when you navigate using the navigation buttons. On the front page I have built a slideshow of cards (that I created using Photoshop). On the build page I have created a preview feature, when you click the option you would like to display on the card the preview section updates. I have included a form and once the form is submitted another section pops up to confirm the order. The order number rises with each order.
To view the code please visit the Github repository.
